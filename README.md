## ğŸ“Œ Tentang Project

**Web Academy** adalah sebuah **online course platform berbasis web**  
yang memungkinkan pengguna untuk **membuat, mengelola, dan mengikuti kelas secara online**.

Pengguna dapat melakukan **enrollment** ke kelas tertentu, dan **akses terhadap materi pembelajaran hanya diberikan kepada pengguna yang telah terdaftar (enrolled)**.  
Platform ini menerapkan konsep **class-based learning** dengan kontrol akses konten.

Project ini terinspirasi dari platform pembelajaran online seperti **Coursera** dan **Mango Network**, namun dikembangkan sebagai versi ringan untuk keperluan pembelajaran dan pengembangan sistem.
